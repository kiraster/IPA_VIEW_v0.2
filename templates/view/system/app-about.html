开发作者: Kiraster

版本: 0.2

发布日期: 2024-09-05

代码框架: 基于pear-admin-flask Python后台管理系统进行二次开发

功能描述: 基于SNMP轮询的IP地址管理检测

实现逻辑: 通过SNMP协议轮询交换机设备，获取ARP，MAC地址，端口类型，端口索引，系统名称，子网掩码等信息后进行逻辑判断处理

感谢: pear-admin-flask Python 后台管理系统开源项目及其贡献者

使用说明

首页

首页

  • 顶栏
    • 已知IP地址统计：显示当前数据库所有IP地址记录
    • 网段统计：显示当前数据库所有网段记录
    • 在线IP地址：显示当前数据库所有IP地址状态为在线记录
    • 离线IP地址：显示当前数据库所有IP地址状态为离线记录
  • 第二栏
    • 左侧折线图：显示当前数据库所有IP地址状态为在线/离线折线图，刷新间隔为10秒
    • 右侧定时任务: 显示当前定时任务运行状态，刷新间隔为10秒
  • 第三栏
    • 左侧柱形图：以分组名称和网段统计的IP地址在线/离线/空闲状态柱形图
    • 右侧卡片: 占位显示

系统监控

  • 左侧栏
    • CPU和内存使用率：显示当前主机实时CPU和内存使用率，刷新间隔为10秒
    • 磁盘信息：显示当前主机磁盘分区情况，页面载入时获取一次
  • 右侧栏
    • 主机信息：显示当前主机信息，页面载入时获取一次
    • 网络信息: 显示当前主机网络接收和发送流量统计信息，刷新间隔为10秒

IP地址使用

IP地址信息

  • 顶栏
    • 搜索功能：以过滤条件+搜索关键字的形式对数据行进行模糊搜索；当前定义的过滤条件有：IP地址、网段、MAC地址、接入设备、Vlan、交换机地址、用户、描述、分组名称
    • IP地址信息：支持的功能：单行编辑、多行删除、过滤字段显示、导出、打印
      • 单行添加：添加功能禁用，由后台定时任务获取更新数据
      • 单行编辑：修改分组名称、用户、描述（其它字段内容为SNMP轮询获取）
      • 多行删除：勾选左侧复选框后点击删除
      • 过滤字段显示：选择表格显示的字段内容
      • 导出：导出csv或xls文件
      • 打印：打印机打印或虚拟打印机打印到PDF文件
  • 分页器
    • 每页显示的条数：每页显示的条数设置为20（可修改为其他值）

分组

  • 左侧栏
    • 分组树：以分组名称和所属网段显示的树图
      • 当分组下没有网段，不能打开节点
      • 点击节点后处于勾选状态
      • 勾选状态仅用于标识所点击的字节，不具其他功能
      • 默认展开所有节点（修改“spread"属性进行折叠）
  • 右侧栏
    • 饼图：显示所点击节点网段的IP地址状态饼图
    • 方块显示: 显示所点击节点网段的IP地址状态分布情况
    • 网段IP地址信息: 显示所点击节点网段的IP地址表

系统设置

账户与权限

用户管理系统（UMS）和RBAC访问权限控制

权限

  • 权限：显示权限数据表的信息，支持在线对权限数据表的增删改查

角色

  • 角色：显示角色数据表的信息，支持在线对角色数据表的增删改查
    • 内置超级管理员，普通管理员，普通用户三种角色
    • 每种角色对应各自的权限

部门

  • 部门：显示部门数据表的信息，支持在线对部门数据表的增删改查
    • 内置集团总部，分支单位1结构
    • 每个用户可对应一个部门
    • 可删除不使用该功能

用户

  • 用户：显示用户数据表的信息，支持在线对用户数据表的增删改查
    • 内置admin超级管理员用户，默认密码123456
    • 内置operate普通管理员用户，默认密码123456，可删除不使用
    • 内置user01普通用户，默认密码123456，可删除不使用

定时任务

  • 定时任务：显示ap-scheduler任务存储器数据
    • 支持添加触发器为interval和date类型的定时任务
    • 添加任务时，可选项不填写参数，使用config.py文件里定义的默认参数
    • date任务的下一次执行时间小于当前日期将从数据库移除
    • 使用在线添加定时任务，需要在"pear_admin\extensions\tasks\tasks.py"文件预先编写执行函数逻辑

IP分组修改

  • 顶栏
    • 搜索功能：以搜索关键字的形式对分组数据行进行模糊搜索
  • 第二栏
    • 左侧显示分组数据表的信息，支持在线对权限数据表的增删改查
    • ID为1的默认组不可删除
    • 删除非默认组时，该分组下的网段自动移动到默认组
    • 点击表格第三栏穿梭框按钮，在右侧显示分组详情
    • 右侧穿梭框可对网段进行加组和离组操作

应用配置

  • 显示当前应用的默认配置
    • 安全性问题: 在生产环境中，不建议直接暴露配置文件的读写操作
    • 当前项目仅开放snmp轮询参数进行设置

日志

  • 显示日志文件内容

关于

  • 当前页面，包含使用说明和应用信息

登陆

  • 登陆成功后，默认保持7天登陆状态

注册

  • 通过页面注册的用户为普通用户
  • 可通过管理员用户对其进行访问权限控制
